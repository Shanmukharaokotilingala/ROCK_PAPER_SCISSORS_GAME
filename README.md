# Rock Paper Scissors Game

A simple console-based Rock Paper Scissors game built in Python, where the user plays against the computer.

## Features

- Player enters a choice: rock, paper, or scissors
- Computer picks a random choice using Python's `random` module
- Compares both choices and declares the result: win, lose, or tie

## How to run

1. Make sure Python 3 is installed
2. Run:
```bash
   python rock_paper_scissors.py
```
3. When prompted, type `rock`, `paper`, or `scissors` and press Enter
